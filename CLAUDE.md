# DevOpsFlow.io — Context Proiect

## Despre
Portal de servicii DevOps pentru Skynet Hosting SRL (Moldova, IT Park).
Domeniu: devopsflow.io

## Tech Stack
- Next.js 15 App Router + TypeScript
- Tailwind CSS 4 + shadcn/ui
- Framer Motion pentru animații
- MDX pentru blog content
- Docker + K8s pentru deployment
- GitHub Actions

## Structura Proiectului
- /src/app — App Router pages
- /src/components — React components
- /src/lib — Utilities, helpers
- /src/content — MDX blog posts
- /public — Static assets
- /infrastructure — Terraform, K8s manifests, Dockerfiles

## Convenții
- Folosește TypeScript strict
- Componente funcționale cu hooks
- Tailwind pentru styling, nu CSS modules
- shadcn/ui pentru componente UI
- Commits convenționale (feat:, fix:, docs:)
- Limbi: RO și EN (i18n cu next-intl)

## Certificări de Afișat
CKA, CCNP/CCNA, LPIC-1, NSE-5/NSE-4, JNCIS-ENT/JNCIA, MTCNA/MTCWE
